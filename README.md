# AVRO-TSC
This is a alpha-version - please see section [Status](#status) for overview of currently implemented features.

## Installation
`> npm install avro-tsc -g`

## Usage
`> avro-tsc ./schemas ./dist/types/`

Format:
`> avro-tsc $INPUT_DIR $OUTPUT_DIR`
